What I’m good at?

Web development requires a certain skill set. It is also the combined work of a team of people, each with their own area of expertise. There are designers, graphic artists, programmers, tech writers, project managers, etc. Although I have knowledge of other areas, my area of expertise is programming. I am most happy when I take a design or idea and program it into reality. Programming a website requires detailed and deep knowledge about HTML, CSS, JavaScript, Front-end frameworks like React, Vue, libraries like Bootstrap, Tailwind.

HTML5 & CSS3

HTML, alongside CSS are the basic languages used to create and style webpages on the internet. Mastering HTML and CSS is the first step any aspiring web developer should do! One quick way to become proficient with HTML and CSS is to practice as much as possible.

JavaScript

If a website was a car, JavaScript would be the engine under the bonnet. In other words, JavaScript is what powers any modern, responsive website. It is essential for any web developer to have a intimate grasp of JavaScript. It is an evolving programming language with an exciting future.

React.js

React is a JavaScript library for building user interfaces. React makes it painless to create interactive UIs. Design simple views for each state in your application, and react will efficiently update and render just the right components when the data changes.

Bootstrap

Bootstrap is one of the best known library for building fast, responsive sites. It allows you to quickly design and customize responsive, mobile-first sites using a front-end open source toolkit, featuring Sass variables and mixins, responsive grid system, extensive prebuilt components and powerful javascript plugins.

Git VCS & Github

Git is the most widely used version control system. It allows many people to work on the same codebase for a project while tracking all the changes that anyone team member has made. Git has the functionality, performance, security and flexibility that most teams and individual developers need. Github is a online platform that integrates with Git.

VS Code

Every developer requires a good text editor. Some old-school developers use editors like Notepad or VIM. One of the best modern editors is VS Code. It has a large community, loads of extensions and tools, source control, terminal integration and loads more. It is my editor of choice.
